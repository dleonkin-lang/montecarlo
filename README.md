# Случайные точки по фигурам

Простой статический проект для урока в 5 классе. Дети выбирают фигуру в квадрате 10×10, бросают случайные точки и смотрят, сколько точек оказалось внутри фигуры, а сколько снаружи.

Онлайн-версия: https://html-preview.github.io/?url=https://github.com/dleonkin-lang/montecarlo/blob/main/index.html

QR-код для открытия сайта лежит в файле `qr-code.svg`.

## Как открыть

Откройте файл `index.html` в этой папке в любом современном браузере.

Сервер, установка программ и npm не нужны.

## Что демонстрирует программа

Программа показывает эксперимент: точки случайно падают в квадрат 10×10. Каждая точка либо попадает в выбранную фигуру, либо остаётся за её пределами.

## Как использовать на уроке

1. Выберите фигуру.
2. Нажмите одну из кнопок для броска точек.
3. Посмотрите на рисунок и на два счётчика.
4. Сбросьте опыт или выберите другую фигуру.
5. Сравните, как меняется рисунок после 10, 100 и 1000 точек.

## Фигуры

- Косой треугольник.
- Пятиугольник.
- Круг в квадрате.
- Озеро.

## Что означают точки

Зелёные точки попали внутрь выбранной фигуры.

Красные точки попали наружу.
